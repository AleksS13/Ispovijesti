# Ispovijesti

Ispovijesti is a full-stack web application for anonymous confessions where users can share their thoughts, like and comment on posts, add favorites, and receive notifications.

---

## 🚀 Tech Stack

- Node.js
- Express.js
- PostgreSQL
- EJS
- HTML, CSS, JavaScript

---

## ✨ Features

- User authentication (register & login)
- Create anonymous confessions
- Like and comment system
- Favorites
- Notifications
- Admin panel (approve, manage users)
